# Seed default automations

## Goal
Prepare the first high-value workflow rules for new organizations.

## Deliverables
- New form submitted
- Missed call text back
- First-response SLA escalation
- Appointment reminder
- Proposal follow-up
- Review request

## Dependencies
- automation engine and relevant business tables exist

## Implementation Notes
- Start with disabled templates when providers are not configured.

## Acceptance Criteria
- New org bootstrap can create a usable automation starter pack
