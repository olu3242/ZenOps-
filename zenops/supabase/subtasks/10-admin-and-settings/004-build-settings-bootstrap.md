# Build settings bootstrap

## Goal
Initialize a new organization with the required settings and starter records.

## Deliverables
- Bootstrap function or script to create teams, roles, default pipelines, stages, lead sources, and automation templates

## Dependencies
- migrations and seed data complete

## Implementation Notes
- This is the bridge between schema readiness and tenant onboarding.

## Acceptance Criteria
- A fresh org can be provisioned into a usable starter state
