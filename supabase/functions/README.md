# ZenOps Edge Functions

This folder contains starter stubs for the first ZenOps Edge Functions.

## Included
- lead-intake
- missed-call-handler
- first-response-sla
- appointment-reminders
- proposal-followup
- review-request
- opportunity-won

These are scaffolds only. They still need payload validation, business logic, logging, and secret configuration before deployment.
