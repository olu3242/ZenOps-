# Seed role hierarchy

## Goal
Define the role hierarchy template used for new organizations.

## Deliverables
- Prepare org bootstrap script or function to create Owner, Operations Manager, Intake Lead, Intake Rep, Sales Lead, Sales Rep, Marketing Manager, and Provider

## Dependencies
- roles table exists

## Implementation Notes
- Roles are org-specific, unlike system profiles.

## Acceptance Criteria
- A new org can be initialized with a complete role hierarchy
